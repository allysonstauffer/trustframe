# TrustFrame

Evaluating when AI-generated image detectors should and shouldn't be trusted.

## Motivation
Deepfake detectors perform well on clean data, but real-world images are 
compressed, resized, and degraded. This project evaluates how common 
transformations affect model accuracy, confidence calibration, and prediction 
stability — and defines conditions where human review is warranted.

## Built on top of
